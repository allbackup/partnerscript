                                         Installation insrtuctions

1 Open file "script/inc/commonvars.php"
2 Find this strings:
  $partnerid='XXX';
  $partnerpwd='XXX';
  $companyname='XXX';
3 Replace 'XXX' with your ID, password and company name

4 Copy files from "script" directory to your site's root directory.
That's all.
